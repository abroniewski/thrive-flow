# ThriveFlow

A productivity and mindfulness app designed to help users plan their day, track habits, reflect on progress, and gain insights into their routines.

## The Problem

- Most productivity apps focus on task completion but ignore the connection between daily habits and personal wellbeing.
- Users who want to plan their day, track habits, and reflect on progress often need three separate tools.
- There is no lightweight app that combines structured daily planning with mindfulness-oriented reflection in one place.

## The Approach

ThriveFlow combines four core workflows in a single interface:

- **Day planning:** Set intentions and priorities for the day.
- **Habit tracking:** Log and monitor recurring behaviours.
- **Reflection:** Record observations on how the day went.
- **Insights:** Surface patterns across habit and reflection data over time.

## Value Delivered

- Reduces the number of tools needed to manage daily productivity and wellbeing.
- Brings habit data and daily reflection into the same context for richer self-insight.
